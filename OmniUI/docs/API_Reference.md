# OmniUI API Reference

## OmniUI (Core)
- **OmniApp**: Main application entry point with Hot Reload support.
- **JuceWidget**: Bridge container for embedding JUCE components.

## OmniWidgets
- **Button**: Standard push button.
- **Slider**: Horizontal value slider.
- **Dial**: Rotary knob.
- **CodeEditor**: Syntax-highlighting text editor.
- **Calendar**: Date picker.

## OmniAudio
- **AudioGraph**: Directed Acyclic Graph manager for audio processing.
- **AudioMeter**: Visual RMS/Peak meter.
- **Sampler**: Audio playback engine.
- **WavetableSynth**: Synthesis engine.
- **PatchBay**: Matrix routing view.

## Omni3D & XR
- **View3D**: 3D viewport wrapper.
- **Camera**: Perspective camera control.
- **XRView**: Stereoscopic VR view.

## OmniNodes
- **NodeCanvas**: Infinite visual graph editor.
- **Node**: Logic block.

## OmniCloud & AI
- **Auth**: User authentication.
- **Storage**: Cloud file management.
- **LLMClient**: AI chat interface.
- **Vision**: Image analysis.

## OmniDebug
- **Profiler**: Real-time performance metrics overlay.
- **CrashHandler**: Error logging and reporting.
