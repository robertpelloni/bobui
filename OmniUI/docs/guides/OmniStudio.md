# OmniStudio IDE

OmniStudio is the flagship IDE built entirely with the OmniUI framework. It demonstrates the power of the platform by dogfooding every major feature.

## Features
- **Integrated Code Editor**: Syntax highlighting, line numbers (OmniWidgets).
- **Visual Audio Graph**: Node-based audio processing (OmniNodes + OmniAudio).
- **3D Scene Editor**: Live preview of 3D assets (Omni3D).
- **Console & Debugging**: Integrated runtime inspection (OmniDebug).
- **System Integration**: Native menu bars and battery status (OmniSystem).

## Launching
```bash
omni studio
```
